# R3-SoftwareTraining-BenjaminSamson
R3 software training submission
Author: Benjamin Samson

This project is created for R3 software subteam training.
The arduino code is for controlling the speed and direction of the motors of the mini rover.
Arduino uno is the microcontroller used for this project.
A dip switch is used for choosing the drive mode.
A potentiometer is used to control the speed of the motors.
